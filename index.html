Even a 4GB RAM Celeron Chromebook can run my Storms! Follow these steps:

  1. At the bottom right of your Chromebook, select the Time.
  2. Select Settings (the gear icon ‚öôÔ∏è).
  3. On the left sidebar, select Advanced, then Developers.
  4. Next to "Linux development environment," select Turn On.
  5. Follow the on-screen instructions.
    • Tip: For the Disk Size, 2GB or 3GB is plenty for my Programs!
    • Username: Use something short (like victor).
  6. A terminal window will open automatically. Type sudo apt update to start.
